# Bakin Engine Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Support RPG Developer Bakin games by unpacking `data.rbpack`, parsing `dic.txt` tab-separated key-value pairs, translating, and re-injecting.

**Architecture:** `BakinEngine` implements the `GameEngine` trait. The `data.rbpack` is unpacked to reveal `dic.txt` (format: `original\ttranslation`). Extraction reads original keys; injection writes translated values back and re-packs. Placeholder encoding handles Bakin-specific control codes.

**Tech Stack:** Rust, std::fs, async-trait

// src-tauri/src/engines/bakin/mod.rs — add

use crate::engines::GameEngine;
use crate::models::TranslationEntry;
use uuid::Uuid;

pub struct BakinEngine;

/// Extract dic.txt path from unpacked rbpack
/// Returns None if file not found — investigate data.rbpack format first (Task 1)
fn find_dic_txt(game_dir: &std::path::Path) -> Option<std::path::PathBuf> {
    // Try common locations — update after inspecting real game
    let candidates = [
        game_dir.join("dic.txt"),
        game_dir.join("data/dic.txt"),
        game_dir.join("Resources/dic.txt"),
    ];
    candidates.into_iter().find(|p| p.exists())
}

pub async fn extract_dic_txt(
    dic_path: &std::path::Path,
    project_id: &str,
) -> anyhow::Result<Vec<TranslationEntry>> {
    let content = std::fs::read_to_string(dic_path)?;
    let mut entries = Vec::new();
    let mut order: i64 = 0;

    for line in content.lines() {
        let parts: Vec<&str> = line.splitn(2, '\t').collect();
        if parts.is_empty() { continue; }
        let key = parts[0].trim();

        if crate::engines::bakin::skip::should_skip(key) { continue; }

        let encoded = crate::engines::bakin::placeholders::encode(key);
        entries.push(TranslationEntry {
            id: Uuid::new_v4().to_string(),
            project_id: project_id.to_string(),
            source_text: encoded,
            translation: None,
            status: "pending".to_string(),
            context: None,
            file_path: dic_path.to_string_lossy().to_string(),
            order_index: order,
        });
        order += 1;
    }

    Ok(entries)
}

mod tests {
    use super::*;

    #[test]

    #[test]

        let dir = tempfile::tempdir().unwrap();
        std::fs::write(dir.path().join("data.rbpack"), b"").unwrap();
        assert!(detect(dir.path()));
    }

    #[tokio::test]
    async fn test_extract_dic_txt_parses_lines() {
        let dir = tempfile::tempdir().unwrap();
        let dic = dir.path().join("dic.txt");
        std::fs::write(&dic, "こんにちは\t\nさようなら\t\n").unwrap();

        let entries = extract_dic_txt(&dic, "p1").await.unwrap();
        assert_eq!(entries.len(), 2);
        assert_eq!(entries[0].source_text, "こんにちは");
        assert_eq!(entries[0].order_index, 0);
        assert_eq!(entries[1].order_index, 1);
    }
}
